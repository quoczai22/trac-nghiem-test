Import / Export câu hỏi

Bản hiện tại chạy local trên GitHub Pages. Import chỉ thêm vào bộ đề, không thay thế toàn bộ để tránh mất dữ liệu. Backend/cloud/phân quyền để trong demo nghiên cứu sau.

Tạo môn local

Tạo vỏ môn trước, sau đó chỉ cần Import thêm câu hỏi theo từng chương hoặc bài. Dữ liệu chỉ lưu trong trình duyệt hiện tại.

Chưa tạo môn local nào.

Nhập file

Chọn môn, chương, thả một hoặc nhiều file PDF/DOCX/TXT/JSON/JS. Hệ thống parse thành câu hỏi và lưu thành các batch import tạm trong trình duyệt.
