# Future Cloud Backend

Khu vực này để lưu ý tưởng backend/cloud cho hè.

Chưa bật ngay vì GitHub Pages là web tĩnh.

Kiến trúc tương lai:

```text
GitHub Pages
    ↓
Node.js API
    ↓
Cloud Database
    ↓
Storage lưu PDF/DOCX
```

Tính năng:

- Đăng nhập.
- Đăng ký.
- Quên mật khẩu.
- Xác minh email.
- Admin duyệt user.
- Import file an toàn.
- Lưu câu hỏi vào database.
