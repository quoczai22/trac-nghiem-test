# Cơ sở dữ liệu - Demo

Môn này là dữ liệu mẫu để kiểm thử luồng nhiều môn ở Phase 1.

- Có ngân hàng 24 câu ở 3 chương.
- Dùng quiz engine, timer, history và random test riêng.
- Dùng namespace LocalStorage `quiz_subject_co-so-du-lieu-demo_*` nên không đè dữ liệu Hệ điều hành.
- Có thể import/export local từ trang `admin/import-export.html` theo cơ chế Append.

Nội dung câu hỏi chỉ phục vụ demo kiến trúc, không phải tài liệu ôn tập chính thức.
